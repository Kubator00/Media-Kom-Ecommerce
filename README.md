# Media-Kom Ecommerce

📋 My simple e-commerce system.


<h2> 🔲 Frontend technologies: </h2>
<ul>
<li>React</li>
<li>Redux Thunk</li>
<li>Formik with Yup</li>
</ul>

<h2> 🔨 Backend technologies: </h2>
<ul>
<li>Express.js</li>
<li>MySQL</li>
<li>JSON Web Token</li>
<li>Bcrypt</li>
<li>Joi Validation</li>
</ul>
